# flutter_auth



## Getting Started

git clone https://github.com/muffynx/firebase-auth.git
cd firebase-auth

## Install 

flutter pub get

## Set Up Firebase

dart pub global activate flutterfire_cli
flutterfire configure

## Run the App
flutter run

Common Errors & Fixes
npm uninstall -g firebase-tools
npm install -g firebase-tools@13.30.0
